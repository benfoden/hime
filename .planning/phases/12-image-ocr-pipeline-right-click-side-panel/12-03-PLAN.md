|----------|-------------|

| Vision response → state derivation | untrusted OCR confidence/text drives which explicit state is shown (no silent blank, IMG-05) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|

| T-12-07 | Information disclosure / integrity | confident garbage on low/no-confidence OCR | mitigate | `deriveImageEntry` forces an explicit no-text or low-confidence state (D-04 / IMG-05); never returns a blank/populated-without-flag entry. |
| T-12-08 | Tampering | malformed/exotic MIME (svg with script) | mitigate | `needsReencode` routes svg+xml/avif/unsupported to PNG re-encode in the SW (Plan 05), so no raw svg/exotic payload reaches the renderer. |
</threat_model>
