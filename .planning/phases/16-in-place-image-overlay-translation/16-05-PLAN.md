    3. Open a page with text-bearing images (manga panel / Japanese menu / a large signage screenshot). Include at least one LARGE image (long edge > 2048px, exercising the downscale path) and one image with `object-fit: contain` or `cover` CSS if possible.

    10. Skip case: if a blob:/tainted image is present, confirm it shows NO overlay (skipped) rather than mis-placed boxes.
