## Accepted deviation — D-V01

Translation moved off Cloud Translation v2 onto the user's existing LLM translation provider
(user steer, commit `269316a`). Google Cloud Vision does OCR + language detection only; the
Google key needs only the Cloud Vision scope. VIS-01/VIS-02 reconciled in REQUIREMENTS.md to
the shipped architecture. No functional break; all flows wired. Future single-call OCR+translate
provider remains tracked as VIS-F1.

3. Progressive toggle → privacy modal/ack → IntersectionObserver + cost guards → badge + panel ✓
4. Copy from panel works on results from both right-click (P12) and progressive (P13) ✓
