Hotkeys

  • Ctrl+Y — Toggle compose mode
  • Ctrl+Shift+Y — YOLO translate
  • Ctrl+Shift+S — Swap language direction
  • Escape — Cancel compose mode
  • Ctrl+Z — Undo translation

Hotkeys can be customized in Chrome's extension shortcuts (chrome://extensions/shortcuts)
