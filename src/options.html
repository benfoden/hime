Translated Search

Bring your own Brave Search key. Brave Search is metered (roughly $0.005 per query as of Feb 2026) — each test or search counts against your Brave quota.

Image Translation

Bring your own Google Cloud key with the Cloud Vision API enabled — used to OCR text out of images. (The detected text is then translated by your configured translation model above, so no Cloud Translation API is needed.) Metered per call — testing and OCR count against your Google Cloud quota. The key is stored locally and read only in the background worker; it never reaches the page.

Progressive Image Translation

When enabled, images you pause near are automatically sent for OCR and translation. Off by default — enabling uploads page images to external services (see below).

Enable Progressive Image Translation?

When you enable this feature, as you scroll a page, images you pause near are automatically uploaded to external services:

  • Image bytes are uploaded to Google Cloud Vision for OCR (text extraction).
  • The detected text is then sent to your configured translation provider (your selected LLM) for translation.

This happens automatically — you do not need to right-click each image. Your acknowledgement is remembered; this prompt will not appear again.

To stop uploads at any time, toggle this setting off. The per-page budget limits auto-translation to 10 images per page.
